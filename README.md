# Agentic Fitness — Weekly Workout Planner

Mobile-first Next.js app for planning, tracking, and adjusting a weekly training schedule. Curate focus areas, balance energy, and swap sessions from a curated library of workouts.

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to explore the planner.

## Scripts

- `npm run dev` — start the development server.
- `npm run build` — create a production build.
- `npm start` — run the production preview server.
- `npm run lint` — run ESLint.
- `npm run format` — format with Prettier.

## Stack

- Next.js 16 App Router with TypeScript
- Tailwind CSS for responsive, mobile-first styling
- Google Fonts via `next/font` for performance

## Features

- Weekly schedule overview with daily focus, anchor, and energy markers.
- Modular workout library filtered by primary focus.
- Quick actions to add/remove workouts and toggle rest days.
- Auto-adjust focus cadence via energy strategy presets.
- Real-time summaries for total weekly duration and focus distribution.

## Deployment

Optimized for Vercel. Run `npm run build` to ensure a clean production build before deploying.
